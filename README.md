## ✨ Key Features & Capabilities

### 🛡️ Security & Identity Verification
* **Automated ID Validation (OCR):** Integrated **OCR.space API** to scan uploaded IDs, extract text, and verify authenticity without manual review.
* **One-Time Password (OTP):** Implemented 2-Factor Authentication (2FA) via SMS to secure user login and registration.

### 📍 Analytics & Visualization
* **Geospatial Mapping (Hotspots):** Interactive **GeoMap** visualization that tracks where appointments are coming from, helping the LGU identify high-demand areas.
* **Feedback Sentiment Summary:** Aggregates user feedback into concise summaries, allowing admins to gauge public satisfaction quickly.
* **Advanced Reporting:** One-click generation of detailed reports (Daily, Monthly, Departmental) in PDF/Excel formats.

### 🔔 Communication System
* **Multi-Channel Notifications:** Automatically sends **SMS and Email** alerts for:
    * Appointment Confirmations
    * Rescheduling
    * Reminders (24h before)

### ⚙️ Core Functionality
* **Smart Scheduling:** Prevents double-booking logic with real-time slot checking.
* **Status Tracking:** Live updates on appointment status (Pending → Approved → Completed).
